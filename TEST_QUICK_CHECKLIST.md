# Quick Test Checklist - Growth Companion SDLC

A condensed checklist for rapid feature testing.

## ✅ DASHBOARD
- [ ] View loads correctly
- [ ] Statistics cards display (Today, Week, Breaks, Tasks)
- [ ] Weekly activity chart visible
- [ ] Responsive on mobile

## ✅ TASKS
- [ ] Create task with title, priority, due date
- [ ] Edit task (title, priority, due date)
- [ ] Complete/uncomplete task
- [ ] Delete task
- [ ] Filter by priority (All/High/Medium/Low)
- [ ] Filter by status (Active/Completed)
- [ ] Search tasks (if available)

## ✅ NOTES
- [ ] Create note (title, content, category, tags)
- [ ] Edit note
- [ ] Delete note
- [ ] Search notes
- [ ] Filter by category (All/Work/Personal/Ideas/Meeting/Other)
- [ ] Tags display and filter correctly

## ✅ TIMESHEET
- [ ] Clock in with task description
- [ ] Edit current task title
- [ ] Switch task (verify subtask creation)
- [ ] Take Short break (15m) with title
- [ ] Take Lunch break (60m) with title
- [ ] Take Custom break with custom duration and title
- [ ] Edit break title (click badge)
- [ ] Clock out
- [ ] View time history (Daily/Weekly/Monthly/Yearly)
- [ ] Verify breaks show titles in history
- [ ] Verify subtasks show in history
- [ ] Export to CSV
- [ ] Export to JSON
- [ ] Export to Excel (verify break titles in "Break Details" sheet)
- [ ] Save work template
- [ ] Use saved template

## ✅ PROFILE
- [ ] View profile information (name, email)
- [ ] Edit profile name
- [ ] Sign out

## ✅ AI CHAT
- [ ] Open/close chat bubble
- [ ] Send basic message
- [ ] Create task via AI
- [ ] Create note via AI
- [ ] Clock in via AI
- [ ] Clock out via AI
- [ ] Start break via AI
- [ ] Switch task via AI
- [ ] Get timesheet status via AI
- [ ] Chat history persists

## ✅ INTEGRATION
- [ ] Create via AI, edit manually
- [ ] Data persists after refresh
- [ ] Navigation between views works
- [ ] Cross-feature sync works

## ✅ EDGE CASES
- [ ] Network error handling
- [ ] Invalid input validation
- [ ] Large datasets performance
- [ ] Time edge cases (midnight transitions)

---

**Quick Test Time:** ~30-45 minutes  
**Full Test Time:** ~2-3 hours
