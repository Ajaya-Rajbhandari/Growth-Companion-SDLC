# Icon Requirements

To replace the app icons, you need to create the following files:

## Required Files

1. **icon-192x192.png** (192x192 pixels)
   - Android home screen icon
   - Square, transparent background recommended

2. **icon-512x512.png** (512x512 pixels)
   - Android splash screen and high-res icon
   - Square, transparent background recommended

3. **apple-icon.png** (180x180 pixels)
   - iOS home screen icon
   - Square, no transparency needed

4. **icon-light-32x32.png** (32x32 pixels)
   - Browser favicon for light theme
   - Square

5. **icon-dark-32x32.png** (32x32 pixels)
   - Browser favicon for dark theme
   - Square

6. **icon.svg** (optional but recommended)
   - Scalable vector icon
   - Works at any size

## Quick Start

1. Create a 512x512 icon design
2. Use an online tool like https://www.pwabuilder.com/imageGenerator to generate all sizes
3. Replace the files in this `public/` directory
4. Rebuild and redeploy

See `docs/ICON_SETUP.md` for detailed instructions.
